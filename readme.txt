Стартовая тема

== Description ==

Стартовая тема для удобной разработки. Собрана на основе underscores.me и FoundationPress. Включает в себя поддержку Woocommerce и другие плюшки

== Installation ==

1. В административной панели зайдите во Внешний вид > Темы и нажмите на кнопку Добавить новую.
2. Нажмите Загрузить и Выбрать файл. Загрузите архив с темой.
3. Нажмите Активировать и пользуйтесь.
